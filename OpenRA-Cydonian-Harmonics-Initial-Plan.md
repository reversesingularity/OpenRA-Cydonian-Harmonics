Asset Pipeline & Technical Notes

Art: Your Grok Imagine skill for concepts → GIMP/Aseprite (or similar) for sprite sheets and palette work. OpenRA supports PNG sheets natively with metadata. Tutorials exist for 3D-to-sprite workflows if you experiment with Blender/Trellis.2 outputs.
Rules: YAML files define actors via trait composition (very flexible—inherit base behaviors, override/add as needed). Example structure for a custom "Harmonic Emitter" or Cian hero unit is straightforward.
Logic: Lua for Resonance interactions, campaign scripting, dynamic objectives, and AI hints. Visual Studio Code has good OpenRA Lua support.
Advanced: Custom C# traits/DLLs only if you outgrow YAML+Lua (not needed early).
Licensing: Engine is GPL—you must provide source for modifications if distributing binaries. Your original artwork, sounds, YAML data, and Lua can be under your preferred terms (e.g., proprietary or custom license). Create 100% original assets (no long-term use of EA Red Alert sprites). Commercial release is possible as a total conversion; many do it successfully.
